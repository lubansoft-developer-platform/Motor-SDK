motor-ts 的引入

两种方式

  • 方式一 开发版-源码版本 需要 motor-ts+cesium 源码库
git clone motor-ts
按照 motor-ts 文档安装完依赖
yarn link

git clone cesium
按照 cesium 文档安装完依赖
yarn link

cd motor-sdk
yarn link motor-ts
yarn link cesium
  • 方式二 开发板-发布版

    使用 motor-ts 打包的代码, 这种比较简单

  1. 把 motor-ts 打包好的 library.zip 放到本项目根目录
  2. 解压
  3. library 重命名 motor
  4. yarn add ./motor
  5. 拷贝 motor-ts 依赖的静态资源到服务器根目录 (目前先以根目录)

luban-motor-sdk 打好的包 用户如何使用
  1. 下载 luban-motor-sdk 压缩包
  2. 压缩包放到项目目录,解压
  3. yarn add ./luban-motor-sdk 或者 npm install ./luban-motor-sdk
  4. 把 luban-motor-sdk 下的文件夹拷贝到开发的静态服务器的根目录
  5. import { Viewer } from 'luban-motor-sdk'
  6. 即可进行开发了
  7. luban-motor-sdk 名字不可以改变
